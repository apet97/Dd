## GET /v1/workspaces/{workspaceId}/holidays/in-period
**Summary:** Get holiday in specific period

### Path parameters
| Name | In | Type | Required | Default | Description |
|------|----|------|----------|---------|-------------|
| `workspaceId` | path | string | Yes |  | Represents workspace identifier across the system. |

### Query parameters
| Name | In | Type | Required | Default | Description |
|------|----|------|----------|---------|-------------|
| `assigned-to` | query | string | Yes |  | If provided, you'll get a filtered list of holidays assigned to user. |
| `start` | query | string | Yes |  | If provided, you'll get a filtered list of holidays starting from start date. Expected date format yyyy-mm-dd |
| `end` | query | string | Yes |  | If provided, you'll get a filtered list of holidays ending by end date. Expected date format yyyy-mm-dd |

### Header parameters
No parameters.

### Request body
This operation does not accept a request body.
