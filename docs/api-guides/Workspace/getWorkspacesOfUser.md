## GET /v1/workspaces
**Summary:** Get all my workspaces

### Path parameters
No parameters.

### Query parameters
| Name | In | Type | Required | Default | Description |
|------|----|------|----------|---------|-------------|
| `roles` | query | string | No |  | If provided, you'll get a filtered list of workspaces where you have any of the specified roles. Owners are not counted as admins when filtering. |

### Header parameters
No parameters.

### Request body
This operation does not accept a request body.

### Code examples
```bash
curl -X GET "https://developer.clockify.me/v1/workspaces" \
  -H 'X-Api-Key: <YOUR_API_KEY>' \
  -H 'Accept: application/json'
```

```javascript

    'Accept': 'application/json',
  },
});
const data = await response.json();

```

```python
import os
import requests

url = 'https://developer.clockify.me/v1/workspaces'
headers = {
    'X-Api-Key': os.environ['CLOCKIFY_API_KEY'],
    'Accept': 'application/json',
}
response = requests.get(url, headers=headers)
response.raise_for_status()
print(response.json())
```
